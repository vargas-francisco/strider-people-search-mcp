# strider-people-search-mcp

> ⚠️ **Unofficial** — this is a community project. Not affiliated with or endorsed by Strider Technologies.

A focused Model Context Protocol (MCP) server that exposes Strider's people-search
and resume-falsification APIs as agent-callable tools. Connect from ChatGPT (Custom
Connectors), Claude Desktop, Claude Code, or any other MCP client.

Status: in development. See [docs/](docs/) for setup once we cut a release.
